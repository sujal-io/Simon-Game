# 🎮 Simon Game

A fun and interactive memory-based game built using **HTML, CSS, and JavaScript**. This project is inspired by the classic Simon Game and was developed as part of Angela Yu’s Web Development course.

---

## 🚀 Features

* 🎯 Pattern-based memory game
* 🔊 Sound effects for each button
* 💡 Visual animations for user interaction
* 📈 Increasing difficulty with each level
* ❌ Game over detection with restart functionality

---

## 🛠️ Tech Stack

* **HTML** – Structure of the game
* **CSS** – Styling and animations
* **JavaScript (Vanilla JS)** – Game logic and interactivity

---

## 🎲 How to Play

1. Press any key to start the game
2. Watch the pattern shown by the game
3. Repeat the pattern by clicking the buttons
4. Each correct round adds a new step to the sequence
5. The game ends if you click the wrong button
